정규식(Regular Expression) : 문자열에서 특정 데이터를 탐색, 확인 할때 사용

  • RegExp 단어로 많이 표현됨
  • 특정 데이터를 탐색, 확인 , 존재여부를 쉽게 확인하기 위해 만든 규칙
  • 웹 분야에서는 자주 사용하진 않음. password 작성 규칙, 첨부파일에서만 자주 사용
  • 알고 있으면 편리한 기능. 다른 언어에서도 거의 비슷한 형식으로 사용
  • /RegExp/flag : (//) 시작과 종료 구문, RegExp : pattern , flag : 규칙을 도와주는 옵션
  • ('str').math(/ragexp/flag) : str에서 매칭되는 항목들을 객체(배열과 유사)로 변환
  • ('str').replace(/ragexp/, '바꿀 문자열') : 정규 표현식에 매칭되는 글자들을 바꿀 문자열로 변환
  • ('str').split(/regexp/) : 문자열을 정규 표현식에 매칭되는 항목으로 분리하여 배열로 변환
  • (/regexp/).test('str') : 문자열기 정규 표현식 규칙에 맞으면 true , 틀리면 false
  • 정규 표현식 플래그
  • i : 대/소문자 구분 x
  • g : 전체 검색
  • m : 행바꿈도 포함
  • a~zA~Z : 영어 알파벳 / ㄱ-ㅎ가-힗 : 한글 / 0~9 : 숫자
  • . : 모든 문자열(숫자,한글,영어,특수문자,공백) 줄바꿈은 x
  • \d : 숫자 \D : 숫자 아닌것 \s : 공백 \S : 공백 아닌것 \w : 영숫자 \W : 영숫자 아닌것 \특수기호
  • 검색패턴 : | or 패턴 a 또는 b 띄어쓰기 x
  • [ ] : 괄호 안의 문자들 중 하나 (or 처리의 묶음) /abc/ => "abc"를 포함하는 /[abc]/ => a 또는 b 또는 c를 포함하는 /[다~바]/ => 다,라,마,바
  • [^문자] : 괄호안의 문자 빼고 나머지 다 [^abcd] : 'a','b','c','d' 4개 제외한 나머지
  • ^문자열$ : ^(시작)$(종료)
