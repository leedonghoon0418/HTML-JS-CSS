window 메서드 중 실행 시간을 지정하는 메서드

  • setTimeout(function,millseconds); 일정시간 후에 function을 실행시킬 때 사용
  • setTimeout을 할당한 객체를 clearTimeout(할당한 객체) 취소 가능
  • setInterval(callback function, millseconds); 일정 간격으로 지속적 함수를 호출할 때 사용
  • claerInterval(할당받은 객체) 중지 가능

millsecond : 1/1000 => 1초 1000 / 0.5초 500 / 0.3초 300
