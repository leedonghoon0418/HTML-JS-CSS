CSS 적용은 element.style.property 형식으로 접근 가능

  • property 는 자바스크립트의 코드 : -(하이픈) 적용안됨.
  • css 속성의 -(하이픈)을 제거하고 , 카멜 표기법으로 변경
  • Element.setAttribute(name,value) : element에 속성 부여 이미 있는 속성이라면 업데이트 , 없으면 새 속성으로 추가
  • cssName.setProperty(name,value) : css 스타일을 부여할 대상에게 css 프로퍼티를 부여
  • name : css에 대한 속성이름 / value : 속성값
